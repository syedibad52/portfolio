# Portfolio Website

This is my professional portfolio website built with React, Vite, and Three.js.

## Tech Stack

- **Framework**: React
- **Build Tool**: Vite
- **Styling**: CSS
- **Animations**: GSAP
- **3D Graphics**: Three.js / React Three Fiber

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
